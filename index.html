本地电脑部署

  1. 安装git

    下载地址安装后可以在搜索中输入gitbash快速打开命令行。

  2. 检测有没有git

    如果有版本则表示已经有了，如果没有提示，则mac会弹框提示你安装git，一路按同意就行了。

  3. 本地配置SSH并绑定至GitHub

    如果命令行中出现了access right，一般是出现了权限问题，就要查看SSH是否正确添加。

  4. 本地安装node.js

    下载地址安装好后输入：node -v如果有版本则已经成功了

  5. 本地安装Hexo

    git clone -b dev 仓库地址

    git clone -b dev git@github.com:xiaoming7658/xiaoming7658.github.io.git
